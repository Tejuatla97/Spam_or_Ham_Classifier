# 🚀 AI Spam Detector

A Machine Learning web application built using Python, Scikit-Learn, and Streamlit that classifies text messages as **Spam** or **Ham (Not Spam)** in real time.

## 🌐 Live Demo

🔗 https://traintestmlspamham-jbvjk9hqr6td4qidkpfhpf.streamlit.app/

---

## 📌 Project Overview

The AI Spam Detector is a Natural Language Processing (NLP) project that demonstrates how Machine Learning can be used to classify text messages into Spam and Ham categories.

The application allows users to enter any message and instantly receive a prediction along with confidence scores.

### Prediction Categories

* 🚨 Spam Message
* ✅ Ham (Not Spam) Message

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Tejuatla97/Train_test_ML_Spam_Ham.git
```

### Navigate to Project Directory

```bash
cd Train_test_ML_Spam_Ham
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
